HTML

  • The head element contains information about the webpage.
  • The body represents the visible content shown to the user.
  • The meta contains information of the data being used in the webpage.
  • The http-equiv sets an HTTP header for content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Each element can be changed individually by adding a css declaration.
  • When a css file is created seperately you have to use link to connect to the html file.
  • text-align aligns the text to where you desire.

Git

  • git status: Checks what branch we are currently working on.
  • git checkout -b "branch name": creates a branch and switches to it.
  • git checkout: Moves the user to a branch or main
  • git clone: clones the code and adds the content where the file location is at
  • git commit -m: Commits staged content to new snapshot.
  • git add -A: commits all changes.
