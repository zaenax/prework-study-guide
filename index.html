HTML

  • The head element contains information about the webpage.
  • The body represents the visible content shown to the user.
  • The meta contains information of the data being used in the webpage.
  • The http-equiv sets an HTTP header for content.
  • Setting up a class allows to seperate and organize different topics and allows to seperately be different in CSS.
  • HTML is the structure of the website.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Each element can be changed individually by adding a css declaration.
  • When a css file is created seperately you have to use link to connect to the html file.
  • text-align aligns the text to where you desire.
  • When calling a class in HTML it can be styled by css by adding a period before the said class.
  • An asterisks changes the whole webpage.

Git

  • git status: Checks what branch we are currently working on.
  • git checkout -b "branch name": creates a branch and switches to it.
  • git checkout: Moves the user to a branch or main
  • git clone: clones the code and adds the content where the file location is at
  • git commit -m: Commits staged content and allows to dscribe what was done.
  • git add -A: commits all changes.
  • git pull allows to update the main branch when everything is added and commited
  • git push pushes changes to the remote repository.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A for loop allows us to continously run code until a certain condition is met.
  • If else allows us to put logic in our code.
  • Function allows us to call a specific code that has it's own conditions.
